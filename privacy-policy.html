WordPress Style Analyzer Pro

Privacy Policy

Last Updated: January 3, 2026

Overview

WordPress Style Analyzer Pro ("the Extension") is committed to protecting your privacy. This Privacy Policy explains how we handle information when you use our Chrome extension.

✅ Key Privacy Points

  • No Personal Data Collection: We do not collect, store, or transmit any personal information
  • Local Processing Only: All website analysis happens entirely on your device
  • No External Servers: No data is sent to external servers or third parties
  • Complete Control: You can delete all Extension data at any time

Information We Collect

Information We DO Collect:

  • Usage Statistics: Anonymous usage data including:
    • Number of websites analyzed
    • Premium vs. free account status
    • Installation date
    • Analysis completion events
  • Local Storage: The Extension stores the following data locally on your device:
    • Analysis results from websites you've analyzed
    • Your usage count (for free tier limitations)
    • Premium account status
    • User ID (randomly generated, not linked to personal identity)

Information We DO NOT Collect:

We Respect Your Privacy:
  • ❌ We do NOT collect your browsing history
  • ❌ We do NOT collect the content of websites you analyze
  • ❌ We do NOT track your identity or link usage to you personally
  • ❌ We do NOT share any data with third parties
  • ❌ We do NOT use cookies or tracking technologies

How We Use Information

The Extension analyzes websites entirely on your local device. All analysis happens in your browser:

  1. When you click "Analyze This Page," the Extension reads the publicly visible HTML/CSS of the current webpage
  2. Analysis results are stored locally in your browser's storage
  3. No data is sent to external servers
  4. You control when to export or delete results

Usage statistics are stored locally only to:

  • Enforce the 3-website free tier limit
  • Determine if you have premium access
  • Track which domains you've already analyzed (to not count them twice)

Data Storage

All data is stored locally using Chrome's storage API:

  • Location: Data is stored on your device only
  • Access: Only you and the Extension can access this data
  • Deletion: You can clear all Extension data at any time by:
    • Removing the Extension from Chrome, or
    • Using Chrome's "Clear browsing data" feature and selecting "Site settings"

Third-Party Services

This Extension does NOT:

  • Connect to external servers
  • Use third-party analytics services
  • Integrate with social media platforms
  • Include advertising networks
  • Share data with any third parties

Premium Account

If you upgrade to Premium:

  • Payment is processed through Stripe, an internationally recognized payment processor
  • We only receive confirmation of payment to activate premium features
  • We do NOT store credit card information
  • Payment processor's privacy policy applies to payment transactions

Permissions Explanation

This extension requires only the minimum permissions necessary for its core functionality. All data processing occurs locally on the user's device. No personal information is collected, stored, or transmitted. These permissions are used only for the core functionality of analyzing websites. We do not use them to track or monitor your browsing.

The Extension requests the following Chrome permissions:

  • activeTab: Required to analyze the current webpage you're viewing
  • storage: Required to save your analysis results and settings locally
  • scripting: Required to inject analysis code into webpages
  • tabs: Required to open new tabs for domain analysis feature
  • host_permissions (http://*/* and https://*/*): Required to analyze any website you visit

There are two permissions in particular for which we want to give you ease.

  • Access the page debugger backend
  • Read and change all your data on all websites

The extension uses Chrome DevTools Protocol for accurate mobile testing. This requires debugger permission of “Read and change all your data on all websites” but we assure you the extension does not alter anything on your site. All data storage about the website being analyzed is temporary and user-session-based:

  • All analysis data is automatically deleted when users start a new analysis or click the Reset button
  • Data is cleared after successful export to ensure no stale data accumulates
  • No data persists beyond the current analysis session

Children's Privacy

The Extension is not directed to children under 13 years of age. We do not knowingly collect information from children under 13.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by updating the "Last Updated" date and, if significant, by displaying a notice in the Extension.

Data Security

We implement reasonable security measures to protect locally stored data:

  • Data is stored using Chrome's secure storage API
  • No transmission of sensitive data over networks
  • No server-side storage of user data

Your Rights

You have the right to:

  • Access all data stored by the Extension (via Chrome's developer tools or export features)
  • Delete all Extension data (by removing the Extension)
  • Opt out of usage tracking (by not using the Extension)

Compliance

This Extension complies with:

  • Chrome Web Store Developer Program Policies
  • Google's Limited Use Policy for Chrome Extensions
  • General Data Protection Regulation (GDPR) principles
  • California Consumer Privacy Act (CCPA) principles

Consent

By installing and using WordPress Style Analyzer Pro, you consent to this Privacy Policy and agree to its terms.
